# Vrinda Store Annual Sales Dashboard

## Project Overview

This project analyzes the sales performance of Vrinda Store for the year 2022 using Microsoft Excel.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard

## Dashboard Features

- Sales vs Orders
- Sales by Gender
- Order Status
- Sales by Channel
- Top 5 States
- Age vs Gender Analysis

## Key Insights

- Women contributed 64% of total sales.
- Amazon generated the highest sales.
- Maharashtra recorded the highest sales.
- Most orders were successfully delivered.
- Adult women purchased the most products.
